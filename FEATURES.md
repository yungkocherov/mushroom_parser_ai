# Описание фич модели (204 шт.)

## Целевые переменные (не входят в модель)

| Переменная | Описание |
|---|---|
| `report_count` | Сырое количество постов-отчётов за день |
| `mushroom_index` | `report_count / audience_scale` — нормализовано по размеру аудитории |
| `mushroom_index_sm` | `mushroom_index / weekday_scale`, сглажено окном 3 дня — финальный таргет |
| `audience_scale` | Относительный рост аудитории (baseline = 2020, значение 1.5 = аудитория выросла в 1.5 раза) |
| `audience_proxy` | Скользящая медиана просмотров постов за 30 дней |
| `weekday_scale` | Коэффициент дня недели (сб~1.4, вт~0.7) |
| `vk_trend` | Google Trends по запросу "вк" — не используется в модели, сохранён для анализа |

## 1. Сырые погодные данные (Open-Meteo)

| Фича | Описание |
|---|---|
| `temperature_2m_mean` | Средняя температура за день (°C) |
| `temperature_2m_max` | Максимальная температура |
| `temperature_2m_min` | Минимальная температура |
| `apparent_temperature_max` | Макс. ощущаемая температура (с учётом ветра и влажности) |
| `apparent_temperature_min` | Мин. ощущаемая температура |
| `precipitation_sum` | Сумма осадков за день (мм) |
| `rain_sum` | Сумма дождя (мм, без снега) |
| `snowfall_sum` | Сумма снега (см) |
| `snow_depth` | Глубина снежного покрова (м) |
| `precipitation_hours` | Часы с осадками |
| `sunshine_hours` | Часы солнца |
| `shortwave_radiation_sum` | Суммарная солнечная радиация (МДж/м²) |
| `wind_speed_10m_max` | Макс. скорость ветра (км/ч) |
| `wind_gusts_10m_max` | Макс. порывы ветра |
| `wind_direction_10m_dominant` | Преобладающее направление ветра (°) |
| `et0_fao_evapotranspiration` | Эвапотранспирация (испарение с почвы, мм) |
| `relative_humidity_2m` | Средняя относительная влажность (%) |
| `pressure_msl` | Атмосферное давление на уровне моря (гПа) |
| `soil_temperature_0cm` | Температура почвы на поверхности (°C) |
| `soil_temperature_6cm` | Температура почвы на глубине 6 см |
| `soil_moisture_0_to_1cm` | Влажность почвы 0–1 см (м³/м³) |
| `soil_moisture_1_to_3cm` | Влажность почвы 1–3 см |

|---|---|

| `gdd_14d` | GDD за 14 дней |
| `gdd_21d` | GDD за 21 день |

## 13. Паттерн "дождь → тепло"

Грибы активно растут после дождя, когда приходит тепло.

| Фича | Описание |
|---|---|
| `rain{N}d_then_warm` | Были ли осадки за N дней (3,5,7,10) и сегодня тепло (>10°C) |
| `good_rain_then_warm_{N}d` | >5мм за N дней (5,7,10) и тепло — более строгий критерий |

## 16. Фурье-сезонность

5 гармоник для описания годовой сезонности с разной детализацией.

| Фича | Описание |
|---|---|
| `sin_doy1`, `cos_doy1` | 1-я гармоника — основной годовой цикл |
| `sin_doy2`, `cos_doy2` | 2-я гармоника — полугодовой |
| `sin_doy3`, `cos_doy3` | 3-я — четвертьгодовой |
| `sin_doy4`, `cos_doy4` | 4-я — ~3 месяца |
| `sin_doy5`, `cos_doy5` | 5-я — ~2.5 месяца (ловит резкие сезонные пики) |

## 17. Пиковые даты и сезонный индикатор

| Фича | Описание |
|---|---|
| `days_to_aug20` | Дней до 20 августа (абсолютное значение) |
| `days_to_sep20` | Дней до 20 сентября |
| `days_to_peak` | Минимум из двух — расстояние до ближайшего исторического пика |
| `is_around_20th_may` ... `_sep` | Окно 18–22 числа в каждом месяце (май–сен) |
| `is_peak_window` | Объединённое окно всех месяцев |
| `season_gauss` | Гауссов колокол с центром doy=240 (конец августа), sigma=60 |
